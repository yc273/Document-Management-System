## 📝 测试步骤

### 1. 安装依赖
```bash
cd D:\软著\文档管理系统\backend
pip install -r requirements.txt
```

### 2. 配置数据库
修改 `app/config.py` 中的数据库密码

### 3. 初始化数据库
```bash
python db_init.py init
```

### 4. 启动应用
```bash
python run.py
```

### 5. 测试接口

#### 注册用户
```bash
curl -X POST http://localhost:5000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"123456","email":"test@example.com"}'
```

#### 登录
```bash
curl -X POST http://localhost:5000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"123456"}'
```

#### 创建文件夹
```bash
curl -X POST http://localhost:5000/api/folder \
  -H "Content-Type: application/json" \
  -d '{"name":"测试文件夹","parent_id":0}'
```

#### 上传文档
```bash
curl -X POST http://localhost:5000/api/file/upload \
  -F "file=@test.txt" \
  -F "folder_id=1"
```

## ⚠️ 注意事项

1. **数据库配置**
   - 使用 SQLite，无需安装数据库服务
   - 数据库文件自动创建在 `backend/doc_management.db`

2. **文件上传**
   - 默认最大100MB
   - 可在 `config.py` 中修改

3. **存储空间**
   - 默认用户1GB
   - 管理员10GB
   - 可在用户管理中修改

4. **局域网访问**
   - 已配置 `host='0.0.0.0'`
   - 防火墙需允许访问

5. **安全建议**
   - 生产环境修改SECRET_KEY
   - 使用HTTPS
   - 限制文件类型和大小
